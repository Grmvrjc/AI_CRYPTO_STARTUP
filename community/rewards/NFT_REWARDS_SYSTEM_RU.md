# Система вознаграждений NFT

## Обзор

Этот документ описывает систему вознаграждений NFT для участников нашего децентрализованного AI-стартапа. Эти NFT являются как признанием вклада, так и дают их владельцам право на долю в будущей прибыли проекта.

## Распределение NFT (всего 1000 NFT)

### Первоначальное распределение
- **420 NFT (42%)** - Зарезервировано для продаж, маркетинга и покрытия непубличных частей проекта
- **580 NFT (58%)** - Выделено для участников сообщества, вносящих вклад в нашу основную цель: создание полноценного MVP и привлечение инвестиций от $1 млн

### Распределение NFT сообщества (580 NFT)
| Категория | Выделение NFT | Описание |
|-----------|---------------|----------|
| Разработка децентрализованной агентской системы | 200 NFT | Для создания прототипа децентрализованной агентской системы, включая инструменты с открытым исходным кодом (смарт-контракты), фреймворки и расширения, использующие нашу ИИ-систему |
| Создание образовательного контента | 200 NFT | Для создания курсов на Степике, инструментов автоматизации управления курсами (боты для синхронизации уроков с GitHub на Степик и общения со студентами), медиаматериалов, проверки уроков и переводов |
| Дизайн, SMM и маркетинг | 100 NFT | Для дизайнеров, менеджеров социальных сетей, маркетологов и менеджеров проектов/курсов |
| Участие сообщества | 40 NFT | Для наиболее активных и интересных участников сообщества и курсов |
| Создание коллекции NFT | 40 NFT | 20 NFT за создание коллекции и 20 NFT за создание изображений |

## Стоимость вознаграждения

- Каждый NFT изначально оценивается примерно в $50
- Когда проект достигнет $1 млн инвестиций, каждый владелец NFT получит $100 (или эквивалент в нашем токене)
- После достижения инвестиционного рубежа условия улучшатся, что позволит немедленно оплачивать работу, в том числе через наш ИИ-эскроу-сервис на TON

## Структура распределения прибыли

Владельцы NFT будут получать процент от чистой прибыли в соответствии со следующими ставками (пока общая сумма выплат на один NFT не достигнет указанной суммы):

| Доля прибыли | Пока выплата не достигнет |
|--------------|---------------------------|
| 42% | $1,000 на NFT |
| 23% | $2,000 на NFT |
| 16% | $5,000 на NFT |
| 15% | $10,000 на NFT |
| 8% | $50,000 на NFT |
| 4% | $100,000 на NFT |

## Дополнительные преимущества для владельцев NFT

- Скидки или бесплатный доступ к нашим ИИ-агентам и другим будущим сервисам в экосистеме
- Расширенный функционал и приоритетный доступ к новым функциям
- Уникальные роли в сообществе
- Другие эксклюзивные преимущества

## Как участвовать и получать NFT

1. **Выберите задачу**: Найдите подходящие задачи в нашем [GitHub-репозитории](https://github.com/LNDMN/AGIENT) или в [Telegram-чате](https://t.me/AI_CRYPTO_STARTUP)
2. **Согласуйте детали**: Обсудите сроки, объем работ и предполагаемое количество NFT с кураторами или администраторами
3. **Выполните задачу**: Код, дизайн, аналитика, контент - всё, что продвигает проект к его цели
4. **Получите NFT**: Мы резервируем за вами необходимое количество NFT и формализуем их после выпуска коллекции
